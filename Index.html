Proyectos

Solo Trainer

Para los jugadores de baloncesto que entrenan solos, es difícil crear rutinas variadas y
eficientes. La aplicación SoloTrainer ofrece entrenamientos personalizados y aleatorios,
adaptados al tiempo y lugar del usuario.

Conocer más

Problema que resuelve: Facilita la creación de rutinas de entrenamiento de baloncesto, personalizándolas según el tiempo y el lugar, ideal para jugadores que entrenan en solitario.

Tecnologías clave: Frontend con HTML, CSS, JavaScript (con React).

Qué demuestra: Habilidad en el desarrollo de aplicaciones web interactivas y manejo de lógica de programación para generar contenido dinámico.

Recetas en tu heladera

¿Qué cocinas cuando no tienes todos los ingredientes de una receta?
Esta aplicación sugiere platos creativos basándose en los ingredientes que tienes disponibles, eliminando el desperdicio de alimentos y facilitando la cocina diaria.

Conocer más

Problema que resuelve: Ayuda a los usuarios a aprovechar los ingredientes que tienen en casa, reduciendo el desperdicio y la indecisión al cocinar.

Tecnologías clave: Frontend con HTML, CSS, JavaScript (con React o Vue.js). Backend con Python (para la IA).

Qué demuestra: Capacidad para integrar tecnologías de frontend y backend, así como conocimientos básicos de inteligencia artificial y manejo de bases de datos.

Duolingo LSA

La falta de herramientas interactivas y accesibles para aprender el Lenguaje de Señas Argentino (LSA).
Este proyecto crea una plataforma de aprendizaje similar a Duolingo, pero enfocada en LSA.

Conocer más

Problema que resuelve: Proporciona una plataforma de aprendizaje interactiva para el Lenguaje de Señas Argentino, contribuyendo a la inclusión y accesibilidad.

Tecnologías clave: Frontend con HTML, CSS, JavaScript (con React). Backend con Python (visión por computadora).

Qué demuestra: Habilidades avanzadas en el desarrollo de aplicaciones web, integración de tecnologías de IA, y un enfoque en proyectos con impacto social.
